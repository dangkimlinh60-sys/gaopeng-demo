## 快速开始

### 本地运行

1. 克隆项目
```bash
git clone https://github.com/dangkimlinh60-sys/heshuangshuang-demo.git
cd heshuangshuang-demo
```

2. 使用本地服务器打开
```bash
# 使用Python内置服务器
python -m http.server 8000

# 或使用Node.js的http-server
npx http-server
```

3. 在浏览器中访问 `http://localhost:8000`

### 部署说明

本项目是纯静态网站，可以部署到任何支持静态网站托管的平台：
